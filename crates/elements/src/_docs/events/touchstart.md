The `touchstart` event fires when the user starts touching an element.

Event Data: [`TouchData`](crate::events::TouchData)

### Example

```rust, no_run
# use freya::prelude::*;
fn app(cx: Scope) -> Element {
    render!(
        rect {
            width: "100",
            height: "100",
            background: "red",
            ontouchstart: |_| println!("Started touching!")
        }
    )
}
```
